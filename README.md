# FRUIT REPORT PROJECT
> This project (run.py) reads through text files containing fuit details and upload them with their corresponding image to an external IP address.
A PDF report (report_email.py) is also created (and sent via email) containing details of the fruits uploaded.\
> 
> NOTE: Created for Python Learning purposes
